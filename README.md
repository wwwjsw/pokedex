## Installation.

### IOS.
- run `yarn` to install dependencies.
- run `npx pod-install ios` to link cocoapods dependencies.
- run `yarn ios` to run on IOS emulator.

### Android.
- run `yarn` to install dependencies.
- run `yarn android` to run on android device or emulator.

## Technical requirements.
